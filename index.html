How to use Logic Lab

Building (Edit mode)

  • Add components: drag them from the left panel onto the worksheet. Gates use IEC symbols (&=AND, ≥1=OR, =1=XOR, 1=NOT/buffer; a circle on the output = inverted).
  • Wires: drag from one pin to another pin (output → input). Wires run with 90° bends — drag any segment sideways to route it around components (dragging the end segments adds extra bends).
  • Buses & tri-state: a Tri-state Buffer passes its data through when the enable input is high, and goes high-impedance (Hi-Z) — drawn as a dashed amber wire — when disabled. Hold Shift while dropping a wire onto an input to add a second driver, so several tri-state outputs can share one wire (a bus). Enable only one driver at a time; if two disagree the status shows ⚠ SHORT CIRCUIT.
  • Junctions: drag a Junction (bus tap) onto the sheet to make a shared bus point. Wire several tri-state outputs into the dot (they merge automatically — no Shift needed), then wire from the dot to whatever reads the bus. The dot's colour follows the resolved bus value.
  • Gate inputs: select a gate, then use the − / + buttons above it to remove/add input pins (2–8).
  • Mux & coders: the Multiplexer, Demultiplexer, Encoder and Decoder resize with the same − / + buttons — MUX/DEMUX step 2:1→4:1→8:1→16:1, and the encoder/decoder grow their address width to match. Data pins are D, select/address pins are S/A, outputs are Y. The encoder is a priority encoder (highest set input wins).
  • LED Matrix: a row/column display — pins along the left are rows, pins along the bottom are columns, and each LED lights when both its row and column line are high. Select it to get two − / + pairs above it (rows on the left, cols on the right). Drive it with a counter/decoder to scan it.
  • Rename: double-click an Input or Output to rename it.
  • Delete: right-click a component or wire for a menu (Delete, Rotate, Rename, Look inside), or select it and press Delete.
  • Rotate: right-click a tri-state buffer and choose ↻ Rotate 90° to point it in any direction — the enable pin stays on the side of the triangle.
  • Selecting several: hold Shift and drag on empty space to draw a box around components (or Shift+click to toggle individual ones). Drag any selected component to move the whole group; press Delete to remove them all.
  • Quick-add: right-click empty space and hover I/O or Logic Gates to drop any component straight from the menu.
  • Palette: click a category heading to collapse or expand it (Latches & flip-flops and Registers & counters start collapsed).
  • Copy / cut / paste: select component(s) and use Ctrl+C / Ctrl+X / Ctrl+V. Pasted inputs/outputs get a suffix ((1), (2)…) if their label is already in use.
  • View: mouse-wheel zooms, drag empty space to pan.

Custom components (ICs)

  • Build a circuit using Input / Output components for the pins, then press 📦 Create IC. The inputs/outputs (top-to-bottom) become the chip's pins.
  • Your chips appear under My Components — drag them out like any other part, and use them inside new chips.
  • Double-click a chip to look inside it (read-only). In edit mode it opens in place (use the breadcrumb bar to go back).
  • Side-by-side inspector (sim mode): drag the handle on the left edge of the worksheet to pull out a second canvas like a curtain. Double-click any chip while simulating to show its insides there — so you watch the inside and the outside live at the same time. The inspector pans (drag), zooms (wheel), drills deeper (double-click a nested chip), and you can toggle its inputs by clicking them. Drag the divider to resize, double-click it (or press Esc) to close.
  • ⬇ / ✕ on a custom component exports / deletes it. Use toolbar Export/Import to move components between sessions as JSON files.

Simulating

  • Click ▶ Simulate. Click Input switches on the sheet to toggle them, or use the Inputs ▾ menu (works even while you are inside a chip) — each input has a toggle and a momentary ⏼ hold button that flips back when released.
  • ▶ Run runs the clock at the slider frequency. Next ⏭ toggles the clock by half a cycle manually, ⏮ Prev steps back through history (also undoes input toggles).
  • All Clock components output the same global clock signal.
  • ƒ boxes next to gate outputs: click to see the live boolean expression for that signal — green letters are currently high, and the popup follows the simulation.
  • Truth table ▾ enumerates the worksheet inputs (up to 8). Boolean ▾ shows the formula for every Output. Timeline ▾ opens a timing diagram of CLK and all inputs/outputs — untick signals to hide them.
  • In expressions: an overline = NOT, · = AND, + = OR, ⊕ = XOR, prev = a feedback signal (stored value), 0 = unconnected pin.
  • 74HC595: like the real chip — shift data in on SHCP edges, copy to the outputs with an STCP edge. Tie MR' to High (1) for normal operation (it is an active-low reset); OE' high forces the outputs low.

Saving

  • 💾 Save stores the worksheet and your components in the browser (localStorage); it is loaded automatically next time.
  • Export/Import JSON files to keep custom components permanently or share them.
